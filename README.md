# Appalachian Trail Data Processing System (C++)

This program reads data from a file containing Appalachian Trail states, miles, and shelters.

It stores the data using structs, sorts the states alphabetically, and calculates total miles and shelters.

## Features
- File input validation (handles empty or missing files)
- Data processing using structs
- Sorting algorithm (alphabetical order)
- Formatted output display

## How to Run

1. Compile:
   g++ main.cpp -o main

2. Run:
   ./main

3. Enter filename:
   data.txt

## Test Files
- data.txt → sample dataset
- empty.txt → used to test empty file handling
